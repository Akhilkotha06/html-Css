3.What are the different formats in which colors in HTML can be declared?

The color of an element can be defined in the following ways:

Built-In Color(h1 {color: color-name;})
RGB Format(h1 {color: rgb(R, G, B);})
RGBA Format(h1 {color:rgba(R, G, B, A);})
Hexadecimal Notation(h1 {color:#(0-F)(0-F)(0-F)(0-F)(0-F)(0-F);})
HSL(h1 {color:hsl(H, S, L);})
HSLA(h1 {color:hsla(H, S, L, A);})
